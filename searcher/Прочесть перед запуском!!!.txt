Запуск программы через searcher.bat

Где -depth - глубина поиска для краулера (когда 1 парсит только начальные узлы), seeds.txt - начальные узлы (можете переименовать файл или добавить ещё узлы построчно).
А text.db имя базы данных (вы можете взять из папки __pycache__ чистую (textclear.db) и переименовать как вы хотите)


Необходимо установить Python 3.3.0 (если не установлен). После установки зайти в Компьютер->Свойства->Дополнительные параметры системы->Переменные среды. Там в поле "Переменные среды пользователя для ..." создать переменную с именем "CLASSPATH" (если ещё такой нет), значением ".;"
Далее, в "Системные переменные" найти Path и добавить путь к папке Python (что-то типа C:\Program Files\Python\). Установка завершена.
