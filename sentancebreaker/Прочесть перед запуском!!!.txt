Запуск программы через breakdowner.bat

roots.txt, prefixes.txt, suffixes.txt, ends.txt - название файлов содержащих морфемы (можно ввести другие названия ваших файлов, только вбивать морфемы по одной на строчку).
А mbase.db имя базы данных (вы можете взять из папки __pycache__ чистую (mbaseclear.db) и переименовать как вы хотите) или mbase.db (с начальной базой).
words.txt - файл со словами, которые нужно разобрать. 

Необходимо установить Python 3.3.0. После установки зайти в Компьютер->Свойства->Дополнительные параметры системы->Переменные среды. Там в поле "Переменные среды пользователя для ..." создать переменную с именем "CLASSPATH" (если ещё такой нет), значением ".;"
Далее, в "Системные переменные" найти Path и добавить путь к папке Python (что-то типа C:\Program Files\Python\). Установка завершена.
